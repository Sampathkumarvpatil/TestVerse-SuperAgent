Getting Started

  1. 1Download ZIP
    AITestversesuperagent.zip
  2. 2Install Python 3.11+
    Download from python.org, add to your PATH.
  3. 3Set Up Virtual Environment
    python -m venv .venv, then activate: .venv\Scripts\activate
  4. 4Install Dependencies
    pip install PySide6 playwright
    python -m playwright install chromium
  5. 5Run the App
    python start.py
  6. 6API Key
    Contact Dr. Sanjeev Kuwadekar or Dr. Sampath V. Patil for your key.
    Paste this key in the app Settings (gear icon).
Terminal: Run python start.py
Paste API key under Settings
Tips for a Smooth Experience:
  • Note: If a site or dropdown fails, intervene manually. SuperAgent will handle the rest.
  • For best accuracy, run your tests in smaller phases.
  • If anything gets stuck during execution, restart the session.
  • Don't edit/view compiled modules; you don't need to touch .pyd files.
  • The app is local—your data never leaves your computer.
